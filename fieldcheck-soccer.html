QUEUED · LOWEST PRIORITY

Soccer, if there's a gap.

Soccer queues lowest among the four sports. The NWSL ecosystem is already well-served by existing data providers and journalism — we'd only build soccer if a specific gap appears that current tools don't address. This page is honesty, not a sales pitch.

§ I · Why we're cautious

Soccer is already well-served.

Unlike volleyball — where we found a real information gap that the platform fills — soccer's data ecosystem is mature. Building here would be redundant unless we identify something specific the existing tools miss.

// The honest assessment

Capability layers should fill real gaps.

Soccer data is mature. FBref, Soccerway, Transfermarkt, Wyscout, StatsBomb already serve professional data needs. The Athletic, ESPN FC, and dedicated NWSL journalism serve narrative needs. U.S. Soccer's national team programs already track development from youth onward. If we build soccer, it has to add something none of these provide — and after looking, we haven't yet found that gap. The honest answer is: we don't build for completeness, we build for usefulness.

§ II · The existing ecosystem

Who already does this well.

A non-exhaustive map of current soccer-data players. If you're researching the NWSL or U.S. soccer pathway, these are the names that come up.

// DATA · PRO
FBref, Transfermarkt
Free comprehensive stats for NWSL, European leagues, national teams. Probably the best free soccer data layer in the world.
// DATA · CLUB-LEVEL
Wyscout, Opta, StatsBomb
Paid professional scouting tools. NWSL clubs already subscribe; scouts use them. The market is filled.
// JOURNALISM · NWSL
The Athletic, ESPN FC
Mature NWSL journalism layer. Multiple full-time reporters. Per-team beat reporters. Sourcing is well-established.
// PIPELINE · YOUTH
ECNL, GA, U.S. Soccer DA
U.S. youth pathway tracked by ECNL national rankings, GA leagues, U.S. Soccer Development Academy. Aggregation already exists.

§ IV · If we build, when?

The trigger conditions.

We'd build soccer only if one of the following conditions emerges. These are the questions we ask before greenlighting any new sport.

// CONDITION 1 · PARENT NEED
A pathway honesty gap.
If parents navigating ECNL → DA → NCAA → NWSL find the existing recruiting services as opaque as volleyball's once were, that's a real signal. Honest pathway probabilities are our specialty.
// CONDITION 2 · NWSL CHANGE
A league structural shift.
If the NWSL adopts equity-for-founding-athletes like LOVB did, or restructures its junior pipeline integration, that's the kind of platform-relevant moment that justifies us building.
// CONDITION 3 · COACH GAP
A coach-development vacuum.
If NWSL or U.S. Soccer launches anything resembling LOVB's "Coach of Coaches" framework, we'd want to surface it independently. That's the kind of integration story we describe well.
// CONDITION 4 · USER ASK
Direct request volume.
If enough volleyball platform users say "I'd use this for soccer too," we'd reconsider. Currently the volume is low. But it's a real possible trigger.
§ V · Tell us your trigger

Let us know what's missing.

If you're working in U.S. soccer — coaching, scouting, parenting, journalism — and you see a specific gap our framework could fill, we want to hear about it. That's the input that triggers a real build.

Tell us the gap. If you see one, we'll listen. File a dispute with the suggestion.
