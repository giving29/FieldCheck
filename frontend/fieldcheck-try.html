// HOW IT WORKS

The framework under the hood.

Five layers of evidence. Eight canonical facets. Sixteen voices. One polygon. One number.

5-layer evidence

Identity · Pathway · Refinement · Resilience · Opportunity. Wikipedia, school roster, conference stats, scout dossier, open-web. Triangulated before grading.

8 canonical facets

Character · Mindset · Mental Strength · Talent · Physical · Mental/IQ · Coachability · Competitiveness. Every sport, same eight. The polygon shape across facets is the verdict.

16 coach voices

Simulated voices from program archetypes (Saban-style, Kerr-style, Pop, Belichick). Each voice weighs facets differently. Disagreement is signal.

Composite + polygon + anchors

A scout sees three things: composite (one number), polygon (8 facets visualized), decision anchors (1–3 specific actions tied to evidence). That's enough to make a call.

// WHY THIS IS DIFFERENT

The moat: depth at the long tail.

Anyone can grade Cooper Flagg. The hard part is grading an athlete the world hasn't noticed yet.

Auto-sport-detect

Scout types just a name. Framework probes Wikipedia + public sources, identifies the sport, runs the right eval grid. No sport-picker round-trip.

Dynamic school resolution

116-school D2/D3 athletics-site map + DDG dynamic resolver + KV cache. Say "Anderson University" — we find athletics.anderson.edu automatically.

Source-cache resilience

External APIs fail. Wikipedia bounces. Conference sites timeout. We cache every successful fetch for 5-21 days so verdicts stay stable as APIs degrade.

Pre-registered predictions
