GET /verdict

Read an athlete

Returns the full FieldCheck IQ verdict: composite + tier + 5-layer moat + comparable + audit envelope. The same payload that renders on /run.

// PARAMETERS
slugstring · reqAthlete slug — e.g. harper-murray-nebraska-womens-volleyball. Use /lookup first if unknown.
include_barsboolInclude numerical scores per moat layer (Physical, Mindset, Production, Pathway, Defense). Default true.
include_auditboolInclude the audit envelope (citations, SHA-256, reasoning trace, drift flags). Default true.
CURL 
# Read Harper Murray verdict, full audit included
curl "https://api.fieldcheck.app/v1/verdict?slug=harper-murray-nebraska-womens-volleyball" \
  -H "Authorization: Bearer fc_edge_xxxxxxxxxxxxxxxxxxx" \
  -H "Accept: application/json"
RESPONSE · 200 
{
  "athlete": { "name": "Harper Murray", "sport": "womens-volleyball", "school": "Nebraska", "position": "OH" },
  "iq_score": 85,

  "moat": {
    "physical": 8.6, "mindset": 9.1, "attacking": 8.4, "defense": 7.9, "pathway": 8.7
  },

GET /comparable

Find the founding-athlete comparable

For any athlete, returns the top-5 nearest founding-athlete archetypes by cosine similarity in the embedded archetype space.

// PARAMETERS
slugstring · reqAthlete slug.
limitintNumber of comparables. Default 5, max 20.
min_matchfloatFloor on similarity. Default 0.65.
JAVASCRIPT 
const resp = await fetch(
  'https://api.fieldcheck.app/v1/comparable?slug=harper-murray-nebraska-womens-volleyball',
  { headers: { Authorization: `Bearer ${process.env.FC_EDGE_TOKEN}` } }
);
const { comparables, audit } = await resp.json();
// comparables: [{ name, match: 0.873, archetype, sources }, …]

GET /trust-graph/walk

Walk the relationship graph

Starting from any node — coach, program, or athlete — walk N edges and return the connected subgraph with citation count per edge.

// PARAMETERS
nodestring · reqNode slug (e.g. coach-john-cook, program-nebraska-wvb).
depthintEdge traversal depth. Default 2, max 4.
edge_typesstring[]Filter — e.g. coached, recruited, transferred.
POST /dispute

Submit a dispute · 5-day SLA

Any verdict, anytime. Methodology desk responds within 5 business days with either a correction (with SHA-256 of new verdict) or a published rationale for the original.

REQUEST BODY · JSON 
{
  "verdict_sha256": "a4f9b2e1c7d8…",
  "dispute_type": "factual_error",
  "claim": "Murray's 2023 hit% listed as .328, official is .331",
  "evidence_urls": ["https://huskers.com/stats/2023"],
  "submitter_org": "Nebraska Athletics"
}
// EVERY RESPONSE IS AUDIT-SIGNED
