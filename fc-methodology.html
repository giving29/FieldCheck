// §1 · THE IQ COMPOSITE

One number. Five layers behind it.

The IQ Score is a weighted geometric mean of the 5-Layer Moat dimensions. Geometric (not arithmetic) so a single floor-level dimension can't be papered over by spikes elsewhere — the lowest layer drags the whole composite down.

// COMPOSITE FORMULA
IQ = round( (Physicalw₁ · Mindsetw₂ · Productionw₃ · Pathwayw₄ · Defensew₅)1/Σw × 10 )
where each dimension ∈ [0, 10] and Σw = total weight

The confidence interval (the ± value next to every score) is a function of citation density per dimension. More citations → tighter band. Few citations → wider band. The math is published in §6.

// §2 · THE 5-LAYER MOAT (SPORT-AWARE)

Five layers. Sport-aware. Position-aware.

Each sport — and within football and baseball, each position — gets a slightly different layer 3 label. The framework is constant: Physical, Mindset, [Production-axis], Pathway, Defense. The production layer name changes because what "scoring" means is sport-specific.

LayerDefault weightWhat it measures
Physical0.22Verified anthropometric + athletic-test data with citations
Mindset0.22Composure, coachability, situational decision-making (qualitative + cited)
Production (sport-aware)*0.30Output that matters in this sport at this position
Pathway0.14Realistic next-step trajectory from current node in the system
Defense0.12Defensive contribution per sport's definition (full per-sport label below)

* The production-layer label per sport: football=PRODUCTION, mbb/wbb=SCORING, baseball.hitter=HITTING, baseball.pitcher=COMMAND, soccer=ATTACKING, mvb/wvb=ATTACKING, tennis=COMPETITION, golf=TOURNAMENT.

// §4 · COMPARABLE MATCHING

The founding-athlete comparable.

Every athlete is matched against the Founding Athletes archetype set — 50 to 200 athletes per sport whose archetype was hand-curated and whose career trajectories are fully traced. Matching is cosine similarity in a normalized 5-layer embedding space.

// COSINE SIMILARITY
match = cos(θ) = (A · B) / (||A|| · ||B||)
where A = the athlete's normalized 5-layer vector and B = the founding archetype's vector

A match ≥ 0.85 is reported as a "strong comparable." A match between 0.65 and 0.85 is reported as a "near comparable." Below 0.65, we return the top-5 with a flag warning the user that no strong archetype was found.

// §5 · NIL SCORE

NIL valuation, with the trace exposed.

The NIL Score is the only score we publish with a complete reasoning trace — every input variable, every weight, every regional adjustment. We do this because in NIL, opacity is a feature elsewhere and a problem here.

// NIL SCORE FORMULA
NIL = 0.32·Social + 0.41·Performance + 0.27·Market
all components ∈ [0, 10], market-adjusted by region. Trace returned with every API response.
// §6 · DRIFT HANDLING

What we do when the truth moves.

Every verdict has a signed_at timestamp. Every input has its own freshness metadata. When a sourced field changes upstream (e.g. a roster page updates), we flag drift in the next request to that verdict and recompute. Confidence intervals widen automatically when stale fields are detected.

// §7 · AUDIT ENVELOPE

The receipt that ships with every verdict.

Every response carries an audit envelope: SHA-256 of the payload, citations list with source URLs, signed-at timestamp, drift flags, methodology version, reasoning trace URL. Walk it. Re-run it. Dispute it. See the API docs for the field-level schema.
